UX/UI

The user experience is a make or break when visiting a website. Which is why I aspire to create a user interface that is interesting and fun, to give the user a great experience. This includes animations, graphics and a visual identity that reflects the audience.

As a student with a frontend focus and programming as the core, I use HTML, CSS and Javascript to create modern web solutions. With the use of Javascript I create dynamic solutions, that takes the final product to the next level, and increases the user experience.
